# Simulation-Homberg

MATLAB/Simulink-Modelle für Fahrdynamik-Simulationen.

## Modelle

| Datei | Beschreibung |
|-------|-------------|
| `TannenwegFinal.slx` | Streckenmodell Tannenweg (finale Version) |
| `SpeedOptimisedVersionTannenweg.slx` | Geschwindigkeitsoptimierte Variante des Tannenweg-Modells |
| `workingGT3nomasknwithcontrol.slx` | GT3-Fahrzeugmodell mit Regelung |

## Voraussetzungen

- MATLAB/Simulink
- Vermutlich benötigte Toolboxen: Simscape oder Vehicle Dynamics Blockset, Control System Toolbox
